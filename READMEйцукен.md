# PlatformixNew — корпоративный лендинг IT-интегратора

Корпоративный сайт для IT-компании. Адаптивная вёрстка, интерактивные разделы, форма заявки.

**Живой сайт:** http://n916005m.beget.tech/

## Что реализовано

- Адаптивная вёрстка — корректно отображается на мобильных, планшетах и десктопе
- Интерактивная статистика с переключением по месяцам — без перезагрузки страницы
- Анимации появления элементов при прокрутке через IntersectionObserver API
- Форма заявки с валидацией полей и обратной связью для пользователя

## Стек

HTML5, CSS3 (Grid, Flexbox, CSS Variables, Animations), Vanilla JavaScript (ES6+), IntersectionObserver API

## Запуск

Открыть `index.html` в любом браузере. Зависимостей и сборки нет.
